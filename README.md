# 🚀 VPNApp

Modern iOS VPN client built with Swift & NetworkExtension

---

## ✨ Preview

![VPN UI](https://images.unsplash.com/photo-1614064641938-3bbee52942c7?q=80&w=400)

---

## ⚡ Features

- 🔐 IKEv2 VPN (native iOS)
- ⚡ One-tap connect / disconnect
- 🌍 Server selection + ping
- 🧠 Auto-connect & Kill Switch
- 🔑 Secure Keychain storage
- 💰 StoreKit 2 (premium-ready)

---

## 🏗 Tech Stack

- Swift
- SwiftUI
- MVVM
- NetworkExtension
- StoreKit 2

---

## 🚀 Run

1. Create SwiftUI project in Xcode  
2. Paste files from repo  
3. Enable capabilities:
   - Personal VPN  
   - Network Extensions  
4. Run on real iPhone
